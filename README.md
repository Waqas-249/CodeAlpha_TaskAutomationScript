# 📂 CodeAlpha - Task Automation with Python

## Project Description

This project automates the task of moving all `.jpg` files from one folder to another using Python.

## Features

- Automatically detects JPG files
- Creates the destination folder if it doesn't exist
- Moves all JPG files
- Displays the names of moved files
- Shows the total number of files moved

## Technologies Used

- Python 3

## Concepts Used

- os
- shutil
- File Handling

## How to Run

1. Create a folder named `source`.
2. Put some `.jpg` files inside it.
3. Run:

```bash
python move_jpg_files.py
```

4. The files will be moved to the `destination` folder.
